# 💜 Quizzy

> Uma aplicação web interativa de perguntas e respostas (Quiz) construída com React e integrada à API do Open Trivia Database,
> que permite também que o usuário crie os seus próprios quizzes.

## ✨ Funcionalidades

* **Autenticação :** Sistema de cadastro e login de usuários com persistência de sessão.
* **Partidas Personalizadas:** O usuário pode configurar sua partida escolhendo o tema, a dificuldade e a quantidade de perguntas.
* **Quizzes autorais:** O usuário pode criar seus próprios quizzes.
* **Perguntas Dinâmicas:** Integração em tempo real com a API pública [Open Trivia DB](https://opentdb.com/), garantindo perguntas variadas e embaralhadas a cada nova partida.
* **Feedback em Tempo Real:** Indicadores visuais imediatos (verde para acerto, vermelho para erro) ao responder uma pergunta, além de um botão de desistência.
* **Resultados Inteligentes:** Tela de conclusão com mensagens, ícones (Troféu, Medalhas, Estrela) e cores customizadas, calculadas dinamicamente com base no percentual de aproveitamento do jogador.
